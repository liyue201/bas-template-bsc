# Deploy

## Quick start

Compile genesis
```
make build-genesis
```

Compile bas 
```
make build-bas
```

Run services
```
make dev-run
```

Show services 
```
make dev-ps
```

Show logs
```
make dev-logs
```

Destory services
```
make dev-down
```
